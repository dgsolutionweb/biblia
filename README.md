# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/drive/1hQwmxhEkPTN1j2pq4EQGhr9wHRn7v3cz

## Run Locally

**Prerequisites:**  Bun


1. Install dependencies:
   `bun install`
2. Set `VITE_GEMINI_API_KEY` in `.env.local` with your Gemini API key
3. Run the app:
   `bun run dev`

## PWA

- No navegador, clique em "Instalar app" na barra de endereços.
- Em iOS, use o compartilhamento e "Adicionar à Tela de Início".
